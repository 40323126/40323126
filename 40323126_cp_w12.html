fix.bat如何做?

1.下載2015課程裡的fix.zip


2.將下載好的fix.zip解壓縮後，打開SciTE


3. 使用SciTE將start.bat及fix.bat開啟


4. 到google搜尋leo-editor github,下載最新版本


5. 把載好的最新版本壓縮檔丟到fix資料夾中


6.打開SciTE，將fix.bat新增一行copy leo-editor-broke-abbrev.zip C:\Ana3_2015\data\


7.再點start.bat,找到第35行改成pip install leo-editor-broke-abbrev.zip


8.兩個儲存完畢後，點選stop，點選fix.bat，再次點選start，即可完成。
